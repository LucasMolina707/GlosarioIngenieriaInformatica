Inicio Objetivos Metodología

    Bienvenido al Portal Académico

    Este sitio es un recurso educativo para estudiantes, facilitando el acceso a glosarios interactivos por materia. Explora y aprende los conceptos clave de cada curso de manera visual y organizada.

    Objetivos del Portal

    El objetivo general es servir como una herramienta de apoyo para el aprendizaje autónomo.

    • **Facilitar la comprensión**: Proporcionar definiciones claras y concisas de términos académicos.
    • **Apoyar el estudio**: Ofrecer un método interactivo de repaso a través de tarjetas y glosarios.
    • **Mejorar la retención**: Usar elementos visuales y un formato de tarjeta para reforzar el conocimiento.
    • **Promover la accesibilidad**: Asegurar que el contenido sea accesible a todos los usuarios, independientemente de sus capacidades.

    Protocolo Académico

    Nuestro protocolo se basa en una metodología sencilla para maximizar el aprendizaje:

    1. **Exploración**: Navega por las materias usando el buscador o el menú de navegación.
    2. **Estudio**: Selecciona una materia y explora sus grupos de tarjetas.
    3. **Repaso**: Usa el botón "Girar" para ver la traducción y definición de cada término.
    4. **Profundización**: Utiliza el botón "Ampliar" para ver la información completa y copiar las definiciones.
