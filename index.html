Introducción al Portal

Este portal es una herramienta de apoyo académico diseñada para facilitar el estudio y la comprensión de términos clave en diversas materias. Utiliza un sistema de tarjetas interactivas (flashcards) para presentar conceptos, definiciones y traducciones de manera visual y organizada.

Objetivos del Proyecto

Objetivo General

Proveer un recurso educativo centralizado y fácil de usar que permita a los estudiantes repasar y consolidar su conocimiento a través de un glosario interactivo por materias.

Objetivos Específicos

  • Facilitar el acceso rápido a definiciones y conceptos clave de cada materia.
  • Ofrecer una experiencia de usuario interactiva y atractiva que fomente el aprendizaje.
  • Centralizar el material de estudio en un formato digital, accesible y bien estructurado.
  • Implementar funcionalidades de búsqueda avanzada para una localización eficiente de la información.

Metodología (Protocolo Académico)

El contenido de este portal se organiza jerárquicamente: Materias → Ventanas → Grupos → Tarjetas.

  1. Selección de Materia: Elige una materia del menú principal o utiliza el buscador.
  2. Navegación por Ventanas: Dentro de cada materia, el contenido puede estar segmentado en "ventanas" o pestañas temáticas.
  3. Exploración de Grupos: Cada ventana contiene grupos de estudio, que agrupan tarjetas relacionadas.
  4. Interacción con Tarjetas: Cada tarjeta presenta un término. Puedes girarla para ver su traducción y definiciones, o ampliarla para acceder a más detalles y herramientas como copiar el texto al portapapeles.

Selecciona una materia usando el buscador para comenzar.
